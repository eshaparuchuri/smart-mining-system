# Smart Mining Digital Twin System

## Overview
This project simulates a Digital Twin for mining equipment using AI.

## Features
- Real-time sensor simulation
- LSTM-based anomaly detection
- Multi-machine monitoring
- Predictive forecasting
- Interactive dashboard (Streamlit)

## Architecture
- Backend: Flask API
- AI Model: LSTM (TensorFlow)
- Frontend: Streamlit Dashboard

## Machines Simulated
- Excavator
- Drill
- Truck

## Run Locally

### Start Backend
python backend/api.py

### Start Dashboard
streamlit run dashboard/app.py
